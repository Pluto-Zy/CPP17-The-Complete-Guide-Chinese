## 第 8 章    其它语言特性

C++17 对 C++ 核心语言进行了一些细小的修改，本章将描述这些修改。

### 8.1    嵌套命名空间

这一特性在 2003 年首次提出，C++ 标准委员会最终允许像下面这样指定嵌套命名空间：

```c++
namespace A::B::C {
  ...
}
```

这等价于：

```c++
namespace A {
  namespace B {
    namespace C {
      ...
    }
  }
}
```

注意（目前）不支持嵌套 `inline` 命名空间。这只是因为目前并不确定应该将 `inline` 应用于最后一个命名空间还是所有命名空间（这两者同样有用）。
